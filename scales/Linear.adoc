/**
 * Setter for the stacked mode.
 * @detailed Some series can be set to stack on top of each other. You can use two stacked mode: "value" and "percent".
 * Stacked mode by "value" is calculated as: the sum values of the previous and the current series is the initial value
 * for the drawing current series. This value is a border of the series stack.
 * Stacked mode by "percent" is calculated as for mode "value", but each point of data draws with a relative percentage
 * to all the points in the same category.<br/>
 * Recommend to use stacked mode for the Bar, Column and Area series.
 * @example anychart.scales.Linear.stackMode_set
 * @param {(anychart.enums.ScaleStackMode|string)=} opt_value ["none"] Value to set.
 * @return {anychart.scales.Linear} Self instance for method chaining.
 */
